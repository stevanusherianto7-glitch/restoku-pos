# Plan: Per-Item 5-Stage Cook Tracker (KDS + CustomerView)

## Konteks (koreksi user 2026-07-16)
Alih-alih tracker per-order, SETIAP menu item dalam pesanan punya progress
5-tahap sendiri:
  dikonfirmasi >> sedang dimasak >> selesai masak >> siap sajikan >> selesai
Tombol advance 1 step per klik. Ilustrasi user: meja A1/A5/A7 →
3x nasi goreng, 2x ayam penyet, 2x es jeruk — masing-masing item tracker mandiri.
